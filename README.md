# 📖 About EchoLink

EchoLink is a real-time emergency communication platform developed for disaster situations where traditional communication infrastructure becomes unavailable.

The platform enables people to communicate, coordinate rescue operations, broadcast emergency alerts, share live locations, exchange important files, and stay connected through an intuitive emergency dashboard.

Built as a **Hackathon Project**, EchoLink demonstrates how technology can help restore communication during catastrophic events.

---

# 📌 Problem Statement

> **"Humanity's first contact with extraterrestrial life didn't begin with war—it began with silence."**

Within hours of arriving on Earth, an alien EMP disabled every form of digital communication. Mobile networks collapsed, internet services disappeared, satellite communications failed, and traditional messaging platforms became completely unusable. With the world's communication infrastructure destroyed, humanity faced its greatest challenge—not a lack of technology, but a lack of connection.

The challenge was to design and develop a next-generation communication platform that enables people to communicate, coordinate, and stay connected when conventional communication systems no longer exist.

---

# 💡 Our Solution

**EchoLink** is a decentralized emergency communication platform designed to restore communication during catastrophic situations.

Instead of relying on traditional communication systems, EchoLink creates a collaborative emergency network where survivors and rescue teams can exchange information in real time.

The platform provides:

- 💬 Real-Time Messaging
- 🚨 Instant SOS Emergency Broadcasts
- 📍 Live GPS Location Sharing
- 🗺️ Interactive Emergency Map
- 📎 File & Image Sharing
- 🎙️ Voice Message Support
- 📊 Live Network Statistics
- 👥 Nearby User Discovery

EchoLink transforms isolated individuals into a connected emergency response network, helping communities coordinate rescue efforts, share critical updates, and improve disaster response when conventional communication systems fail.

> **Our Mission:**  
> *Reconnect Humanity When Traditional Networks Fail.*

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/<your-username>/EchoLink-v2.git
```

### Navigate to Project

```bash
cd EchoLink-v2
```

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
npm start
```

### Open in Browser

```
http://localhost:3000
```
